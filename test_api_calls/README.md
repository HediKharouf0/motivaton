# GitHub API Test

Minimal Node.js app that authenticates with GitHub OAuth and displays your commits.

## Prerequisites

* **Node.js** ≥ 18 (ships with global `fetch`)

## Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Start the server
npm start
```

Then open [http://localhost:3000](http://localhost:3000) — **everything else is done from the browser**:

1. The app walks you through creating a GitHub OAuth App (with the exact values to fill in)
2. You paste your **Client ID** and **Client Secret** into the web form
3. The credentials are saved to `.env` automatically
4. Click **Login with GitHub**, authorize, and you're in
5. Pick a repo from the dropdown to see commits

## Dev mode (auto-reload)

```bash
npm run dev
```

## API Endpoints (local)

|Method|Path|Description|
|-|-|-|
|GET|`/api/config/status`|Check if OAuth is configured|
|POST|`/api/config`|Save Client ID \& Secret to `.env`|
|GET|`/auth/github`|Start GitHub OAuth flow|
|GET|`/auth/github/callback`|OAuth redirect handler|
|GET|`/auth/logout`|Clear session|
|GET|`/api/me`|Current user info|
|GET|`/api/repos`|List your repositories|
|GET|`/api/commits/:owner/:repo`|List commits for a repo|
